# ChatBot

Este projeto é um ChatBot desenvolvido em Python.

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Lhrds/ChatBot.git
   ```
2. Execute o arquivo `chatbot.py`:
   ```bash
   python chatbot.py
   ```
